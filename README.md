### 步骤

1. 修改capture.py文件下的cap中bbox的范围

   ```python
   img = ImageGrab.grab(bbox=(100, 290, 950, 1000)) #参数依次为x，y，宽，高，一般把视频放在左边顶上容易调整
   ```

2. 修改image_search.py 中的APP_IP等参数

   ```python
   """ 你的 APPID AK SK """
   APP_ID = '自行申请appid'
   API_KEY = '自行申请api key'
   SECRET_KEY = '自行申请secret_key'
   ```

   ​

3. 启动Search.py，运行调试截图的时候可以将start方法注释，节省api次数

   ```python

   #截屏 获取地址
   path=capture.cap()
   # 开始识别
   start(path)
   ```
